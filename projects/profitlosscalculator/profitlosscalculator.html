This is profit loss calculator app which will calculate your profit or loss and percentage when provided with

  1. Initial Price
  2. Quantity of stocks
  3. Current Price

While building this app I learned

  1. Taking input from the user by using Document.querySelector() method in DOM
  2. Use of event listner by using .addEventListener
  3. Use of loops inside a function
